# Gulp Planner

Gulp Planner is a powerful abstraction layer for Apache Flink that allows you to define stream processing jobs using simple YAML configuration files. It provides a declarative way to specify complex stream processing pipelines without writing Java code.

## Key Features

- **Declarative Job Definition**: Define your entire Flink job using YAML configuration
- **Rich Transformation Support**: 
  - Window Operations (Tumbling, Sliding, Session)
  - Aggregations (Sum, Count, Average, Min, Max)
  - Field Transformations
  - Filtering and Mapping
  - Key-based Operations
- **Cloud-Native Integrations**:
  - Amazon Web Services (S3, Kinesis, MSK)
  - Google Cloud Platform (BigQuery, Pub/Sub, GCS)
  - Azure (Event Hubs, Blob Storage)
  - Apache Kafka
- **State Management**: 
  - RocksDB State Backend
  - Filesystem State Backend
  - Memory State Backend
- **Type-Safe Operations**: Built-in schema validation and type checking

## Project Status

This project is actively maintained and built for Apache Flink 1.20 with JDK 17 compatibility.
